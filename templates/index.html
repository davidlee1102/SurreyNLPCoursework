Embedding
Embeds the input words into a 100-dimensional vector space. Input length is set to the maximum sequence length.
Conv1D
A 1D convolutional layer with 32 filters, a kernel size of 3, same padding, and a ReLU activation function.
MaxPooling1D
A 1D max pooling layer with a pool size of 2, reducing the spatial size of the convolutional layer output.
LSTM
A Long Short-Term Memory layer with 128 units to model temporal dependencies in the data.
Dense
A fully connected output layer with 14 units and a softmax activation function for multi-class classification.
